Welcome!

Welcome to the USCIS GitHub.com environment! The GitHub.com service has been selected to be a Pilot as a Source Code Management (SCM) tool for multiple teams within the USCIS. Each team will use repositories within GitHub.com to manage and store both their source code and supporting documentation. By using GitHub.com we look to increase efficiency and promote a more collaborative environment throughout each team’s project lifecycle.

This page will cover some basics and provide you with links to self-guided training and help documentation regarding different features and common practices within GitHub.com. To view all of the help and training documentation please visit the Help and Training Repository.

User Account Requirements

In order to maintain good user visibility and keep the organization running smoothly a few requirements have been put in place for users regarding their accounts.

• Incorporate your name in your handle
Allows other users to easily identify who is making requests and comments (e.g. JSmith25)

• Upload a picture of yourself as your profile picture
Allows users who don’t know you to put a face to the name

• Associate your account to your USCIS email address
(e.g. John.Smith@uscis.dhs.gov)

• Enable 2FA
For instructions read the Two-Factor Authentication Overview located in the Self Study document in the Help and Training Repository.

Support

If you are having issues with GitHub.com please use the Issues feature to create a support request in the Support repository.
